# 常见问题与异常处理

## 应用数据迁移暂停

**问题现象1**

在数据加载界面，应用数据迁移暂停。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/a9/v3/FRNEdA5RTveAt6FBctEa4w/zh-cn_image_0000002529714125.png?HW-CC-KV=V1&HW-CC-Date=20260312T113831Z&HW-CC-Expire=86400&HW-CC-Sign=0452CFAC2905E22C7A1EC9C73DEA717A0F793EF1C0E0FFB7069BD17D355A280C)

**可能原因**

应用数据迁移的过程中需要使用到网络，当前终端设备网络不可用，导致数据迁移暂停。

**解决方法**

单击“稍后连接WLAN加载”按钮，进入桌面后连接网络，终端设备网络可用后，恢复应用数据迁移。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/fa/v3/EDUsRf29Scq7cbSgJuAxjA/zh-cn_image_0000002529714129.png?HW-CC-KV=V1&HW-CC-Date=20260312T113831Z&HW-CC-Expire=86400&HW-CC-Sign=27F8C393A1938E2B8B44E28A8BCB5E2C783578CFCBED916BE1876A5857271BA2)

**问题现象2**

进入桌面之后，若双升单数据迁移还未结束，可通过通知栏进入应用加载界面查看加载进度

在应用加载界面，应用数据迁移暂停。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/a2/v3/gmam9cGwRXuZGboFZ5ugeQ/zh-cn_image_0000002529714127.png?HW-CC-KV=V1&HW-CC-Date=20260312T113831Z&HW-CC-Expire=86400&HW-CC-Sign=FA02752F67FC9C2F09BC47C243FB902E6C28780B7D729A973061DDBC21970126)

**可能原因**

应用数据迁移的过程中需要使用到网络，当前终端设备网络不可用，导致数据迁移暂停。

**解决方法**

单击“稍后连接WLAN加载”按钮，进入桌面后连接网络，终端设备网络可用后，恢复应用数据迁移。

## 应用数据迁移执行十五分钟后失败

**问题现象**

应用数据迁移执行十五分钟后显示失败。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/20/v3/ATe3hnCxQt6qf_u6g419pg/zh-cn_image_0000002497754188.png?HW-CC-KV=V1&HW-CC-Date=20260312T113831Z&HW-CC-Expire=86400&HW-CC-Sign=8196C01787990BABB892DD80F1565D5EFB27F748E0F3D132CE02BA518CF678D6 "点击放大")

**可能原因**

单个应用数据迁移执行超过十五分钟，超过设定的单个应用最长数据迁移时间，任务执行失败。

**解决方法**

请优化应用BackupExtensionAbility的代码实现，在十五分钟内完成应用数据迁移。

说明

已接入“数据迁移框架”的应用完成数据迁移后，才可以被消费者使用。尽可能快的完成应用数据迁移，可以带给消费者更好的体验。
