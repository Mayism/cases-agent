# 申请开放能力权限指导

## 开放能力申请准备

请先参考[应用开发准备](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/application-dev-overview)完成基本准备工作，再继续以下开放能力准备项。

### 室内高精度定位

为了更好的用户体验，系统侧对室内高精度定位服务功能做了权限保护处理，使用相关接口开发者需先提交“室内高精度定位”能力开关的申请，在申请通过后，再使用该能力。

1.  登录[AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html)，选择“开发与服务”。
2.  在项目列表选择项目，并在应用列表下选择需要申请室内高精度定位功能的应用。
3.  进入“项目设置 > 开放能力管理”页面，选择能力名称为定位服务（HarmonyOS NEXT），然后点击“室内高精度定位”对应的“申请”。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/b8/v3/1FVxwoSnSIq0r1lFFL393g/zh-cn_image_0000002517106130.png?HW-CC-KV=V1&HW-CC-Date=20260324T110133Z&HW-CC-Expire=86400&HW-CC-Sign=2131E62D0E4A8F1EC708F50F0A654C837C80B656DBDF0922B20090ECC10AE5DD "点击放大")
    
4.  参考“申请原因”中的模板，提供申请必需的相关信息，包括应用介绍、使用场景、申请用途，然后点击“提交”按钮。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/97/v3/32oP9cnxSsi5KQjXpBtBdg/zh-cn_image_0000002530645649.png?HW-CC-KV=V1&HW-CC-Date=20260324T110133Z&HW-CC-Expire=86400&HW-CC-Sign=B07D0F0778828D70CACF409CB2D9A5328482985ADDB82D198E8758DBBC5C439E "点击放大")

返回“开放能力管理”页面，原“申请”变为“申请中”，1~3个工作日内反馈申请结果，请留意互动中心的“服务开通申请”信息。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/b4/v3/X2R3VaSSQke0sW0cEqSrPw/zh-cn_image_0000002498925726.png?HW-CC-KV=V1&HW-CC-Date=20260324T110133Z&HW-CC-Expire=86400&HW-CC-Sign=BF1C9480829C32E1CF2E1AE124A8E5CEC05A71148E88D72A07137D6E90E6BDC5 "点击放大")

申请通过后，互动中心会发送通知给您，同时“申请中”会变为置灰显示的“申请”，至此，应用已成功开启室内高精度定位开放能力。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/b8/v3/7Dex7dsWSveKxzOu45QXyg/zh-cn_image_0000002498765738.png?HW-CC-KV=V1&HW-CC-Date=20260324T110133Z&HW-CC-Expire=86400&HW-CC-Sign=830E2799CA4D580DD565D2DBFD51910708700E66CA30A325EE92E4CE998DC9C2 "点击放大")
