# 华为支付签约的免密代扣在哪里可以解约？

-   方式1：用户可选择“华为钱包 > 我的 > 华为支付”，点击右上角菜单栏“免密代扣 > 华为支付免密代扣”，完成免密代扣解约。
-   方式2：商户（开发者）通过对外开放的[API接口](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/payment-withhold-unsign)进行解约。
