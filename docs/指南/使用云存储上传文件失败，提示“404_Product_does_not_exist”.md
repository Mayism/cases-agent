# 使用云存储上传文件失败，提示“404:Product does not exist”

**问题现象**

使用云存储上传文件失败，HiLog提示“404:Product does not exist”。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/78/v3/iPA688i9Sd-qkQdmqJ-VAg/zh-cn_image_0000002440934292.png?HW-CC-KV=V1&HW-CC-Date=20260312T212833Z&HW-CC-Expire=86400&HW-CC-Sign=ACB6AD0200BF15E966FAFC28F457D874EBD4F5785185A561EFB1C9DCFCD0DFB5)

**解决措施**

云存储服务端返回的错误，出现此错误是因为云存储服务未开通。请[开通云存储服务](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cloudfoundation-enable-storage)。
