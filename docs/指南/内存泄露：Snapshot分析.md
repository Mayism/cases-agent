# 内存泄露：Snapshot分析

针对方舟虚拟机，DevEco Profiler提供了内存快照分析能力，结合Memory实时占用情况，分析不同时刻的方舟虚拟机内存对象占用情况及差异。

注意

由于隐私安全政策，已上架应用市场的应用不支持使用Snapshot分析模板。
