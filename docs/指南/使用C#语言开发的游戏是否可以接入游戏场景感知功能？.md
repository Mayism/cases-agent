# 使用C#语言开发的游戏是否可以接入游戏场景感知功能？

当前提供游戏场景感知C# SDK，可以实现使用C#语言开发的游戏接入游戏场景感知功能，具体请参见[游戏场景感知C# SDK开发指导](https://gitee.com/petal-gaming-services/HarmonyAdaptivePerformanceForUnity/blob/main/游戏场景感知C-SDK开发指导.md)。
