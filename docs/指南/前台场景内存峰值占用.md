# 前台场景内存峰值占用

## 规则详情

应用/元服务前台场景峰值内存占用：应用在前台且亮屏使用规程的内存占用应≤ 1500MB。

## 检测逻辑

1.  执行hdc shell。
2.  执行hidumper --mem <进程pid>命令，获取如图Pss字段。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/1f/v3/7_GOB5z_QwuERu8SVDtrpg/zh-cn_image_0000002500909884.png?HW-CC-KV=V1&HW-CC-Date=20260313T050344Z&HW-CC-Expire=86400&HW-CC-Sign=55FA4B9FAEDEDF4F727B14BECC09F1C309E7FD4F70BB6BFC114BE1419110ADAA)

## 计算逻辑

执行多轮测试，取最大Pss值为占用峰值，内存占用小于1500M。
