# 智慧调优

DevEco Studio的Profiler工具中已集成智慧调优能力，支持通过自然语言交互，分析并解释当前实例或项目中存在的性能问题，帮助开发者快速定位影响性能的具体原因。

从DevEco Studio 6.0.0 Beta1版本开始，CodeGenie支持对Launch冷启动问题和Frame卡顿丢帧问题进行智慧调优分析。

从DevEco Studio 6.0.0 Beta3版本开始，CodeGenie支持对Allocation内存分配问题和Snapshot内存堆快照问题进行智慧调优分析。

从DevEco Studio 6.0.2 Beta1版本开始，CodeGenie增加了OOM内存溢出场景的分析能力，主要包括ArkUI组件、NAPI、闭包等内存问题场景。

## 操作步骤

1.  首次使用请先根据界面提示完成CodeGenie授权登录。当前支持如下两种开启方式：
    
    **方式一：**若Launch、Frame、Allocation、Snapshot模板已录制完成，点击**Session**窗口中该条会话上的![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/2b/v3/XcW1PK2JTtGYtwqfRHnR2A/zh-cn_image_0000002532669891.png?HW-CC-KV=V1&HW-CC-Date=20260313T040858Z&HW-CC-Expire=86400&HW-CC-Sign=C28761210E1B084D882247E983B906F6EB264F86FB7962F52633E870FD8443DF "点击放大")图标，即可开始智慧调优分析。录制方法具体请参考[性能问题定位：深度录制](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/deep-recording)。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/d8/v3/ZtkA3BjjQ-aOcIaDZXFcLw/zh-cn_image_0000002532669899.png?HW-CC-KV=V1&HW-CC-Date=20260313T040858Z&HW-CC-Expire=86400&HW-CC-Sign=EC463DF3423CEBFE05891B10DA77D78007F7A6902DAD75312D110123160DF445 "点击放大")
    
    **方式二：**切换至**Assistant**窗口，点击**Create Session**开始录制调优任务；或点击**Open File**按钮导入已有的调优数据文件，当前支持导入的文件类型包括.insight、.heapsnapshot、.rawheap。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/eb/v3/iB1vyFq9SmWZcTolqFjeTQ/zh-cn_image_0000002532669893.png?HW-CC-KV=V1&HW-CC-Date=20260313T040858Z&HW-CC-Expire=86400&HW-CC-Sign=6A75A2590D9B9345B63B32A177C358A6CD7F77AA9CF811A988530F8E3E9CB3A4 "点击放大")
    
2.  对于方式二，在**Assistant**页面，点击**Create Session**按钮，从**Launch**、**Frame**、**Snapshot**、**Allocation**中选择一个分析模板。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/e9/v3/n3I2tzZxQlK2uv8Q2SSPDA/zh-cn_image_0000002532669897.png?HW-CC-KV=V1&HW-CC-Date=20260313T040858Z&HW-CC-Expire=86400&HW-CC-Sign=51330EF844752CB25289F0F293D85427655CFAD64094E7613EBE7F8718C14E2A "点击放大")
    
    说明
    
    使用Snapshot模板对堆快照问题进行分析时，支持在对话框中选择单个Snapshot分析，或选择两个Snapshot进行对比分析。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/cf/v3/XONV7iZCSkO-dXLe3CcwsA/zh-cn_image_0000002532669889.png?HW-CC-KV=V1&HW-CC-Date=20260313T040858Z&HW-CC-Expire=86400&HW-CC-Sign=44BA9F46B7CA14D1EDA41FD0BBF38C587B8247ED547D0B853A9F1F7EDA2C809E)
    
3.  以Allocation为例，录制新的调优任务或导入本地已有的调优数据模板文件。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/bb/v3/PpVXRUMcS7CMInXZZoKOUg/zh-cn_image_0000002532749841.png?HW-CC-KV=V1&HW-CC-Date=20260313T040858Z&HW-CC-Expire=86400&HW-CC-Sign=4E22B17952394ADA7B13A48EE0E99D505C1E045A05320179AEE6608C124ACD22)
    
4.  等待AI完成初步分析。左键点击高亮的泳道名称，点击**Analyze**进一步分析该阶段的具体内存信息，点击**View Lane**在右侧查看具体的泳道信息。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/0/v3/d8JNgXEvStK38zTHR64Bbw/zh-cn_image_0000002501069816.png?HW-CC-KV=V1&HW-CC-Date=20260313T040858Z&HW-CC-Expire=86400&HW-CC-Sign=CF29B221630071FE75243C7F34D7DBA54ADDDA2D0DBC10C70A9E88412D57BEF3)
    
5.  点击**Analyze**后，逐步深入挖掘当前异常场景，找到影响性能的可能原因。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/01/v3/dQLp7wH4QpK0730BqqOdZw/zh-cn_image_0000002501069824.png?HW-CC-KV=V1&HW-CC-Date=20260313T040858Z&HW-CC-Expire=86400&HW-CC-Sign=EDB7D0A95EFB0DAC58CA0DED7F24671FB990DFC991C082F2B2ED23845B2250C5)
