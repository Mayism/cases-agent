# 多进程调试

部分设备上，UIAbility支持以独立进程的方式运行并调试，详细请参考[进程模型](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/process-model-stage#其他进程类型)，可按照以下步骤对UIAbility进行调试。

## 编译构建配置

1.  新建一个Ability，该Ability继承AbilityStage，作为独立进程的入口。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/18/v3/hi1BuAoHRli2dobu3I6J1g/zh-cn_image_0000002532749903.png?HW-CC-KV=V1&HW-CC-Date=20260313T045442Z&HW-CC-Expire=86400&HW-CC-Sign=96ED226CE87E23DBAFC81216A0F79C7DB5C8F52849A52678AC9DDD04C31FB8E4)
    
2.  右键ets目录，新建其它需要作为独立进程启动的UIAbility。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/63/v3/5nLh6IACSlKHbNLNLFdskw/zh-cn_image_0000002532669947.png?HW-CC-KV=V1&HW-CC-Date=20260313T045442Z&HW-CC-Expire=86400&HW-CC-Sign=CE06F923065F1423B425F749DD2E9258709144E3B719BF5D8AD1CF5B91A77495 "点击放大")
    
3.  修改module.json5配置文件，增加独立进程入口及isolationProcess配置项。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/58/v3/wiiX4zLoQa-f8LPOymlP7Q/zh-cn_image_0000002532749893.png?HW-CC-KV=V1&HW-CC-Date=20260313T045442Z&HW-CC-Expire=86400&HW-CC-Sign=AE70F5994F43CF8EE9B388E992F67073426312862985EA440E6E3B4A7C6E5E91)
    

## 调试

1.  编写跳转UIAbility的代码。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/8b/v3/xQMs_SWEQO2YDaEohoMwtA/zh-cn_image_0000002532749897.png?HW-CC-KV=V1&HW-CC-Date=20260313T045442Z&HW-CC-Expire=86400&HW-CC-Sign=7156BD040A185ADB5997295BA422E1674199DDADFD562A33A87F49CFD0E54645)
    
2.  在跳转的UIAbility中或独立进程入口处设置断点，启动调试。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/b4/v3/ccK-pHg9SoC08ijHhFllOA/zh-cn_image_0000002500910032.png?HW-CC-KV=V1&HW-CC-Date=20260313T045442Z&HW-CC-Expire=86400&HW-CC-Sign=96BD3B46D2BE82490ABB8FD3A2D7A4DE16462E04EF004A70370486D7C37FAFEE)
    
    跳转到以独立进程启动的UIAbility时将会新启动一个调试会话窗口。
