# 创建项目

本章以[Caffe SqueezeNet](https://github.com/forresti/SqueezeNet)模型集成为例，说明App集成操作过程。

1.  创建DevEco Studio项目，选择“Native C++”模板，点击“Next”。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/88/v3/u1h-9c_nSPKDfxNWDSluLQ/zh-cn_image_0000002496568528.png?HW-CC-KV=V1&HW-CC-Date=20260313T012837Z&HW-CC-Expire=86400&HW-CC-Sign=F6234788EBB9519E01084F87A40DC5CCC5B811D62A2F5B319FABFD3EB1838F3F "点击放大")
    
2.  按需填写“Project name”、“Save location”和“Module name”，选择“Compile SDK”为“5.1.0(18)”及以上版本，点击“Finish”。
