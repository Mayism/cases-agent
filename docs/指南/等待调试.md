# 等待调试

开发者可以通过将某个应用设置为“等待调试模式”，然后当开发者需要对应用进行调试时，拉起应用即可快速进入调试。

说明

-   应用设置为“等待调试模式”后，此时如果启动普通的debug调试，将会取消当前的等待调试模式。
-   设置“等待调试模式”前，需要将应用安装到设备上。

## 操作步骤

1.  在设备选择框中选择调试的设备，并单击Run > Attach to Process by Name。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/1d/v3/SJhY-JXTSKGdi_SoNFSF-w/zh-cn_image_0000002500910540.png?HW-CC-KV=V1&HW-CC-Date=20260313T045401Z&HW-CC-Expire=86400&HW-CC-Sign=FB6E834048F2ED05CD435764B72D7B20EBEF2137A52B543EF79C4DBB08ADD2CD)
    
2.  选择需要设置为“等待调试模式”的应用（默认为当前工程），选择需要进行调试的调试类型。然后单击**Attach**，即可将该应用设置为“等待调试模式”。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/89/v3/90YGLW7STrW2TjYSZp0VYQ/zh-cn_image_0000002501070388.png?HW-CC-KV=V1&HW-CC-Date=20260313T045401Z&HW-CC-Expire=86400&HW-CC-Sign=85E549EB26C580F1ADE858B062536C4C960715F287D51D6693EA6E3B851E7378)
    
    此时会在DevEco Studio底部显示一个等待进度条，在应用被拉起之前，将一直处于等待状态。可通过进度条右侧的取消按钮进行取消。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/f8/v3/W3YtnnugSZSzdSNWBMeCaA/zh-cn_image_0000002532750407.png?HW-CC-KV=V1&HW-CC-Date=20260313T045401Z&HW-CC-Expire=86400&HW-CC-Sign=3809010D630C55495684E54D19EA6C3CE88C28AACA1F761ACC8B271E3D65C562)
    
3.  拉起设备端应用，此时将会进入调试。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/1e/v3/e2T1QnS5TSqSEMdgN8pikQ/zh-cn_image_0000002501070386.png?HW-CC-KV=V1&HW-CC-Date=20260313T045401Z&HW-CC-Expire=86400&HW-CC-Sign=07B0D45FA52BD5FAD79ED95B813E36770182676163667B1E7107EA78D0969B09)
