# GetFormat

## 函数功能

获取Tensor的Format。

## 函数原型

```cpp
ge::Format GetFormat() const;
```

## 参数说明

无

## 返回值

| 类型 | 描述 |
| --- | --- |
| ge::Format | 返回tensor的Format值，默认值为FORMAT_RESERVED。 |

## 异常处理

无

## 约束说明

无
