# 打开文件

## 场景介绍

普通应用无法直接访问公共路径下的文件，Enterprise Data Guard Kit为应用提供相关接口以获取文件描述符（fd）。

## 接口说明

详细接口说明可参考[接口文档](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/dataguard-fileguard)。

| 接口名 | 描述 |
| --- | --- |
| [openFile](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/dataguard-fileguard#section132239575117) | 使用Callback方式获取指定目录文件fd。 |
| [openFile](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/dataguard-fileguard#section1292915920212) | 使用Promise方式获取指定目录文件fd。 |
| [openFileWrite](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/dataguard-fileguard#section779510429368) | 使用Callback方式获取指定目录文件只写模式fd。 |
| [openFileWrite](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/dataguard-fileguard#section198141025172920) | 使用Promise方式获取指定目录文件只写模式fd。 |

## 开发步骤

1.  导入模块。
    
    ```javascript
    import { fileGuard } from '@kit.EnterpriseDataGuardKit';
    ```
    
2.  初始化[FileGuard](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/dataguard-fileguard#section18457214114215)对象guard，调用接口openFile或者openFileWrite，并且可选择以下一种方式获取指定目录文件fd。
    
    -   通过回调函数方式，获取文件fd。
    
    ```typescript
    import { BusinessError } from '@kit.BasicServicesKit';
    function openFileCallback() {
      let guard: fileGuard.FileGuard = new fileGuard.FileGuard();
      let path: string = '/data/service/el2/test/test.txt';
      guard.openFile(path, (err: BusinessError, fd: number) => {
        if (err) {
          console.error(`Failed to open file. Code: ${err.code}, message: ${err.message}.`);
          return;
        }
        console.info(`Succeeded in opening file. path: ${path}, fd: ${fd}.`);
      });
    }
    ```
    
    -   通过Promise方式，获取文件fd。
    
    ```typescript
    import { BusinessError } from '@kit.BasicServicesKit';
    function openFilePromise() {
      let guard: fileGuard.FileGuard = new fileGuard.FileGuard();
      let path: string = '/data/service/el2/test/test.txt';
      guard.openFile(path).then((fd: number) => {
        console.info(`Succeeded in opening file. path: ${path} , fd: ${fd}.`);
      }).catch((err: BusinessError) => {
        console.error(`Failed to open file. Code: ${err.code}, message: ${err.message}.`);
      });
    }
    ```
