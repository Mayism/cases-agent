# 单击快速验证手机号按钮，无法拉起页面

**现象描述**

日志报错示例：

```
login fail, error: {"code":1001502014,"message":"The app does not have the required scopes or permissions."}
```

**解决措施**

未申请Scope权限，请参考[申请账号权限](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/account-config-permissions)文档进行操作。
