# 启动和关闭模拟器

在设备管理器页面，单击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/2d/v3/FGY_4mSFQBaH16sSyasy2A/zh-cn_image_0000002500908792.png?HW-CC-KV=V1&HW-CC-Date=20260313T045113Z&HW-CC-Expire=86400&HW-CC-Sign=3D8E4C9EFA6886D86E11B6BFDDBCF26A54AA4B1B6BF499BB260A4BEDBC1D83B2)即可启动模拟器。模拟器启动时会默认携带上一次运行时的用户数据，包括用户上传的文件，安装的应用等。如果是新创建的模拟器，则不会携带用户数据。如果想清除上一次运行时的用户数据，点击**Actions >** ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/a8/v3/EHK_sPhQSvycPxKoR9tNPA/zh-cn_image_0000002500908786.png?HW-CC-KV=V1&HW-CC-Date=20260313T045113Z&HW-CC-Expire=86400&HW-CC-Sign=768ACDEC859A64C399663FD400967FB2032C44C2CE850D67FE27CDDC588C11FB) **> Wipe User Data**。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/ee/v3/PuiHX3mXQEm6ZVCPtR_uGQ/zh-cn_image_0000002532668693.png?HW-CC-KV=V1&HW-CC-Date=20260313T045113Z&HW-CC-Expire=86400&HW-CC-Sign=C5CD52971661268AE1C436758BB865B0F30FDD95B6E614788C5680D3B6B66E66 "点击放大")

在模拟器运行期间，可以点击**Actions >** ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/0d/v3/TBXTifCqQKW34kaTwSPqlg/zh-cn_image_0000002500908782.png?HW-CC-KV=V1&HW-CC-Date=20260313T045113Z&HW-CC-Expire=86400&HW-CC-Sign=FA13D3F7BAF6ACF1EC0AE65C920B1ACB897A5F1C8914364572FE308F6E6FA539) **> Show on Disk**显示模拟器在本地生成的用户数据。点击**Actions >** ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/c9/v3/NjNf4CX9QbimbJH9nvO3VA/zh-cn_image_0000002500908778.png?HW-CC-KV=V1&HW-CC-Date=20260313T045113Z&HW-CC-Expire=86400&HW-CC-Sign=E243634C695F1906387D32CFAED812272F1A23B95BE7ACD85485779AD8C815D7) **> Generate logs**可以生成模拟器自启动到此刻的所有日志信息。想要关闭运行中的模拟器，可以在设备管理器页面点击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/b4/v3/w1X1gkRFRYunjsd5tWTY0w/zh-cn_image_0000002532668661.png?HW-CC-KV=V1&HW-CC-Date=20260313T045113Z&HW-CC-Expire=86400&HW-CC-Sign=0C750F821DCA41BC8B354AB36533FC24725338C328DE17351BD0868FFD09A2A9)，或者点击模拟器工具栏上的关闭按钮![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/a7/v3/ivCM14CIRH6a9BtTQ3bFpg/zh-cn_image_0000002500908798.png?HW-CC-KV=V1&HW-CC-Date=20260313T045113Z&HW-CC-Expire=86400&HW-CC-Sign=3E6D969C2780D89FBCB54E5933C15DA48C620DBE77D7824B26CBA139F9664AB5)。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/47/v3/mKuMZCizQwehOqxcnor5tQ/zh-cn_image_0000002532668665.png?HW-CC-KV=V1&HW-CC-Date=20260313T045113Z&HW-CC-Expire=86400&HW-CC-Sign=5AA2FBC0D7570EB48282772D1FC0E2D4298DA3F3A1E66589D6FE3EF5020260CD "点击放大")

模拟器关闭后，点击**Actions >** ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/5e/v3/M4v3Ez4ATqaFBIX9jO5v_g/zh-cn_image_0000002500908796.png?HW-CC-KV=V1&HW-CC-Date=20260313T045113Z&HW-CC-Expire=86400&HW-CC-Sign=4ADCAACB9A69D4C95AE8702AF6D78D5AE18203ED754357AE347820053268EA4D) **> Delete**可以删除模拟器，并清除模拟器的用户数据和配置信息。
