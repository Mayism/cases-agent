# @since

@since标签表示在特定版本中添加了类、方法或其他符号。

## 语法

@since <versionDescription>

## 示例

使用 @since：

```javascript
/**
 * Provides access to user information.
 * @since 1.0.1
 */
export function UserRecord(): void {}
```
