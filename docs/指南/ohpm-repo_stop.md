# ohpm-repo stop

停止ohpm-repo实例。

## 命令格式

```vbnet
ohpm-repo stop
```

## 功能描述

用于停止已经启动的ohpm-repo实例。

## 示例

执行以下命令：

```vbnet
ohpm-repo stop
```
