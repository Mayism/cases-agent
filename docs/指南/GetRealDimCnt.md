# GetRealDimCnt

## 函数功能

获取TensorDesc所描述Tensor的实际维度个数。

## 函数原型

```cpp
int64_t GetRealDimCnt() const;
```

## 参数说明

无

## 返回值

| 类型 | 描述 |
| --- | --- |
| int64_t | TensorDesc所描述的实际维度个数。 |

## 异常处理

无
