# 剪贴板粘贴框遮挡智能填充选择框

**现象描述**

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/d9/v3/4fxYgtyTQ4qZp6Jn-9yWIg/zh-cn_image_0000002470492194.jpg?HW-CC-KV=V1&HW-CC-Date=20260324T110306Z&HW-CC-Expire=86400&HW-CC-Sign=58BCAB5480DE16775263D74C199125821B9CF4C5F294DF67793A5AF0A7DBFCDB "点击放大")

**解决措施**

在代码文件中设置.selectionMenuHidden(true)，使剪贴板粘贴框隐藏。

```
      Row() {
        Text('收货人：').textAlign(TextAlign.End).width('25%')
        TextInput().width('75%').contentType(ContentType.PERSON\_FULL\_NAME).selectionMenuHidden(true)
      }
```
