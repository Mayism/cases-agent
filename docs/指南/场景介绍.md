# 场景介绍

## 手机与穿戴设备分布式协同

-   重要通知实现腕上提醒，实现即时通知推送，如：在手机侧App中设置日程提醒、用药提醒、任务提醒等数据，可以同步到穿戴设备侧App中，不用打开手机，也可以随时在穿戴设备上查看重要信息。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/a0/v3/yJ15zuCkRC-BGT6wlYAJyw/zh-cn_image_0000002190453285.png?HW-CC-KV=V1&HW-CC-Date=20260312T141924Z&HW-CC-Expire=86400&HW-CC-Sign=205BC6C95FE17C00BEE567390B957AE523D0AD3F92C71A22C3CBD2C8B04B54C2 "点击放大")
    
-   为手机应用带来全新的“腕上交互”，如：用手机侧App看视频、听音频时，可以通过穿戴设备侧App操控手机侧App，比如暂停，下一首，停止等。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/43/v3/_tA04mxsRV-rySnX4IVmOA/zh-cn_image_0000002155046636.png?HW-CC-KV=V1&HW-CC-Date=20260312T141924Z&HW-CC-Expire=86400&HW-CC-Sign=3A0A9620C9A995FDF900F4900A17AF6D9FE8F0DB89C2C1890A812D83B55A64E1 "点击放大")
    
-   手机侧和穿戴侧的实时协同体验。如：在手机侧App开启导航，可以通过穿戴设备侧App提示用户左转、直行、右转等。在走路或骑行时，不用每次拿出手机查看路线。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/7c/v3/vR8-FVoRT12GrZR4Pb8oTg/zh-cn_image_0000002190567613.png?HW-CC-KV=V1&HW-CC-Date=20260312T141924Z&HW-CC-Expire=86400&HW-CC-Sign=1EDD47E61DD13D7FABA61AAAA40345E016CA5EC570D01B8290F580E08AE61424 "点击放大")
    

## 手机与穿戴设备虚拟化

开发者在手机侧开发App，而不需要在穿戴设备侧开发App。

-   穿戴设备状态实时掌握，如：手机侧App实时获取穿戴设备侧连接状态、佩戴状态、电量状态等，给用户提供更多增值服务。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/46/v3/gSGwVqeOQ86mcuGBkbH4pg/zh-cn_image_0000002155206396.png?HW-CC-KV=V1&HW-CC-Date=20260312T141924Z&HW-CC-Expire=86400&HW-CC-Sign=D268F564721594B0BD50828E4C25AC48A70EFA4CB71476824576D8E62E946D5F "点击放大")
    
-   穿戴设备传感器：手机侧App读取穿戴设备的传感器数据（如ECG、PPG、ACC数据等）。
    
    说明
    
    传感器受限发布，仅限专业研究机构使用。
