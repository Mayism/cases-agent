# GetPlacement

## 函数功能

## 函数原型

```cpp
ge::Placement GetPlacement() const;
```

## 参数说明

无

## 返回值

| 类型 | 描述 |
| --- | --- |
| ge::Placement | 返回tensor的Placement值，默认值为kPlacementEnd。 |

## 异常处理

无
