# 部署云侧工程

您也可选择在云函数和云数据库全部开发完成后，将整个云工程资源统一部署到AGC云端。

1.  右击云开发工程（“CloudProgram”），选择“Deploy Cloud Program”。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/35/v3/p5Zn675oQp-jGEojjFW99g/zh-cn_image_0000002214858797.png?HW-CC-KV=V1&HW-CC-Date=20260313T040434Z&HW-CC-Expire=86400&HW-CC-Sign=6B4F21414FD0FB82C433F0251DE7F1BF8A66B115DEEC256540CF550A66867091)
    
2.  您可在底部状态栏右侧查看云工程打包与部署进度。
    
    请您耐心等待，直至出现“Deploy successfully”消息，表示云工程已成功部署。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/9b/v3/21b7W52wT0yJv5OmVhlreg/zh-cn_image_0000002179338472.png?HW-CC-KV=V1&HW-CC-Date=20260313T040434Z&HW-CC-Expire=86400&HW-CC-Sign=F549812947C6144F95BA0E4D96760D8C1AC66E234D1F45E63B2C7C84A87E56BB)
    
3.  在菜单栏选择“Tools > CloudDev”。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/d0/v3/HBuFI5GMRba-Lb5OPGidlA/zh-cn_image_0000002214858789.png?HW-CC-KV=V1&HW-CC-Date=20260313T040434Z&HW-CC-Expire=86400&HW-CC-Sign=B992B5C29EDC716143145FC42170D6013E7CBA9B9871B37065A7EA45746B5A8E)
    
4.  在打开的CloudDev面板中，点击“Go to console”，打开当前项目的AGC云开发子控制台。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/e5/v3/Ms1hnNHBQDaMs-q4FouJmA/zh-cn_image_0000002179498156.png?HW-CC-KV=V1&HW-CC-Date=20260313T040434Z&HW-CC-Expire=86400&HW-CC-Sign=8232311546A43EE2451B6051FE89A479889ABD53130E1CAC13ACF1FA00892E0D)
    
5.  分别进入云函数与云数据库服务菜单，可查看到您刚刚部署的云函数与云数据库资源。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/b8/v3/wtTPVbnASWOW4hNCpDGYLQ/zh-cn_image_0000002467759440.png?HW-CC-KV=V1&HW-CC-Date=20260313T040434Z&HW-CC-Expire=86400&HW-CC-Sign=D460385AB465E672B99445F2C72B3F0CE29634C318AD5AAC0BE2286908C01F57)
