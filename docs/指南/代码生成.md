# 代码生成

CodeGenie具备自然语言代码生成能力，在**对话框内**输入代码需求描述，点击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/21/v3/INqkBbQYQRaAoxNrKXrTtQ/zh-cn_image_0000002535794547.png?HW-CC-KV=V1&HW-CC-Date=20260313T040734Z&HW-CC-Expire=86400&HW-CC-Sign=F380CAA530870B9221706C20407C742583E3340E44CB8AE802CE8A0C3DBAB4B5)发送，将自动生成符合要求的代码段。

DevEco Studio 6.0.1及之前版本，生成的代码一键复制![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/7f/v3/z7Ru4E3mQ-ym1IwLfMrQ6g/zh-cn_image_0000002504392404.png?HW-CC-KV=V1&HW-CC-Date=20260313T040734Z&HW-CC-Expire=86400&HW-CC-Sign=EC8E2290077C7904111572946AE01F13A281BF7E1EBF3AE283EFF839ABE25F7C)或一键插入![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/50/v3/S-ZQdrw0RxOaQO-6MSh7MQ/zh-cn_image_0000002504232574.png?HW-CC-KV=V1&HW-CC-Date=20260313T040734Z&HW-CC-Expire=86400&HW-CC-Sign=81437E96034A38998E9C68F67FB1BE373517C2A195EEF35F886EF998BB42A48A)至编辑区当前光标位置。

在DevEco Studio 6.0.2 Beta1版本，生成的代码直接应用到代码文件中；在**Changed Files**中可查看被修改的文件，修改前后内容对比，逐项接受或拒绝；代码还原；以及支持在问答区编译验证功能。

从DevEco Studio 6.0.2 Release版本开始，使用HarmonyOS Act智能体时，生成的代码直接应用到代码文件中；在**Changed Files**中可查看被修改的文件，修改前后内容对比，逐项接受或拒绝；代码还原；以及支持在问答区编译验证功能。

以DevEco Studio 6.0.2 Release和DevEco Studio 6.0.1 Release版本为例分别说明，示例如下。

**DevEco Studio 6.0.2 Release版本**

**操作步骤**

1.  选择HarmonyOS Act智能体，在对话框输入功能描述，点击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/7d/v3/4C8vJeZkRTuCmmUvT-sZTg/zh-cn_image_0000002518720936.png?HW-CC-KV=V1&HW-CC-Date=20260313T040734Z&HW-CC-Expire=86400&HW-CC-Sign=E8FCEBF2FC76CCF5DB6947E64E8A098E98BD3143A14A474CD8D06584BAE6F9DB)发送，等待生成。
2.  在问答区域的**Changed Files**可以查看被修改的文件，点击文件对比修改前后差异；将鼠标悬浮在文件路径上，点击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/08/v3/uSv4QtZyShmiB-D-ik2nZA/zh-cn_image_0000002550160773.png?HW-CC-KV=V1&HW-CC-Date=20260313T040734Z&HW-CC-Expire=86400&HW-CC-Sign=F241F7CAA4C6DA4870AE602A301C738D1B357FCBA9FA39960BD390267CFBDB8F)可接受或拒绝该文件的修改；点击**Accept All****/Reject All**按钮，接受或拒绝所有文件的修改；在编辑器右键**Local History** > **Show History**，查看历史修改文件还原代码。
3.  点击问答区中**Run**，可以编译验证；开启**Auto Run**开关，可以开启自动编译验证。Auto Run更多描述可参考[Agent配置](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-agent-use#section2075893021715)。

**示例**

在index页面中添加一个可以跳转至另外页面的按钮。
