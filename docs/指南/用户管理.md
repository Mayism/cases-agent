# 用户管理

用户管理页面可以新增用户、修改用户类型、重置用户密码，删除用户和搜索用户，页面效果如下图所示：

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/6b/v3/35jTZqIbRNWVMCUUOAS8jQ/zh-cn_image_0000002501068620.png?HW-CC-KV=V1&HW-CC-Date=20260313T035550Z&HW-CC-Expire=86400&HW-CC-Sign=C8CF65DC199ECAD25F0A862CF379ED43355C2584F60B7FCA13C26523608BEEF1 "点击放大")

1.  点击“新增”按钮，弹出新增用户面板，输入新增用户的用户名和密码，新增用户首次登录将强制重置密码。填完用户信息后点击新增即可添加一个新用户，页面效果如下图所示：
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/71/v3/EBLJt-epRSOq8fYA0WPj_w/zh-cn_image_0000002500908788.png?HW-CC-KV=V1&HW-CC-Date=20260313T035550Z&HW-CC-Expire=86400&HW-CC-Sign=22141EB7249B5917E4C2A236B385FC407CB4879B1094BC51C595CE60C8B48D2F "点击放大")
    
2.  点击“编辑类型”按钮，弹出编辑用户类型面板，在此面板中可以修改用户类型成管理员或用户。点击确认修改用户类型，将出现密码输入框，由于管理员修改其他用户的类型是敏感操作，故需要输入当前操作账户的密码进行再次验证，页面效果如下图所示：
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/4e/v3/01OxFcTyQUydfERqI4VwhQ/zh-cn_image_0000002500908784.png?HW-CC-KV=V1&HW-CC-Date=20260313T035550Z&HW-CC-Expire=86400&HW-CC-Sign=59DC1A8A983B5342AE7C12CBFB27D6D60F46159FB5807DA371D650587F3FD888 "点击放大")
    
3.  点击“重置密码”按钮，弹出重置用户密码面板，在此面板中可以通过点击生成新密码为用户生成随机新密码，并可通过点击复制图标将新密码复制进剪切板（只有点击**确定**按钮才会重置密码）。点击确认重置密码，将出现密码输入框，由于管理员对其他用户重置密码是敏感操作，故需要输入当前操作账户的密码进行再次验证，页面效果如下图所示：
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/2b/v3/pPidPvUnRlmN_hpuO4NdJw/zh-cn_image_0000002532748635.png?HW-CC-KV=V1&HW-CC-Date=20260313T035550Z&HW-CC-Expire=86400&HW-CC-Sign=064828B3A491D12BC9CC06459F0CC88E1E957C86920B43A4C2FD6134AF195159 "点击放大")
    
4.  点击“删除”按钮，弹出删除提示，如果确定删除，需要点击按钮“是”，将出现密码输入框，由于管理员删除用户属于敏感操作，需要输入当前操作账户的密码进行再次验证，页面效果如下图所示：
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/5e/v3/kkMekEjDQr6ogNmNyBrLJw/zh-cn_image_0000002532748629.png?HW-CC-KV=V1&HW-CC-Date=20260313T035550Z&HW-CC-Expire=86400&HW-CC-Sign=D67FB5913A0EFDB9A836E2776F0DC7FBE24DB95698A7E01BBD5681C9BE58A244 "点击放大")
    
    当被删除的用户是某个三方包的唯一所有者时，禁止被删除。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/dd/v3/i9RdXLmXRkCepVoW_E2L4g/zh-cn_image_0000002532748649.png?HW-CC-KV=V1&HW-CC-Date=20260313T035550Z&HW-CC-Expire=86400&HW-CC-Sign=ABC035DDE0911DC656EFA90CCAE781895B42A7BE5A860C950FD29E92BDEA4AEE "点击放大")
    
    5\. 点击搜索框，支持指定用户类型（系统管理员/普通用户）和用户名模糊搜索，搜索页面效果如下图所示(以指定用户类型为系统管理员，用户名为admin为例)：
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/71/v3/Ce1yfw1RS1uScjG94m0tNQ/zh-cn_image_0000002532748639.png?HW-CC-KV=V1&HW-CC-Date=20260313T035550Z&HW-CC-Expire=86400&HW-CC-Sign=F7CCFE968ED14C4C5CD6FCF19EAD9DC9802EE22FB2340D567C376B4A689A0326 "点击放大")
