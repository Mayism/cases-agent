# 代码分析

CodeGenie支持对在**对话框中**输入对代码段和代码文件分析要求，帮助开发者快速理解代码逻辑、代码功能、技术细节和潜在问题等，提升开发效率。

DevEco Studio 6.0.2 Beta1之前版本，分析代码文件时需要通过**Files**入口选中文件；分析代码片段时，选中代码段后需点击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/5e/v3/DSVSqwcwSbuX964wOZphdg/zh-cn_image_0000002550201375.png?HW-CC-KV=V1&HW-CC-Date=20260313T040724Z&HW-CC-Expire=86400&HW-CC-Sign=853C58D0C7CB1D25643F63FB65841574C811EC813A78649C0A335E99E657ABBD)图标开启光标上下文功能。

在DevEco Studio 6.0.2 Beta1版本，分析代码文件时，支持在对话框输入要分析的代码文件或直接分析当前文件；分析代码片段时，选中代码段后直接分析，无需开启![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/56/v3/J19-efcOSCe8S7fx0yMU3w/zh-cn_image_0000002518561638.png?HW-CC-KV=V1&HW-CC-Date=20260313T040724Z&HW-CC-Expire=86400&HW-CC-Sign=C80E64318EFF7D10F0C926E459AFF49A0A5B82C12A36FB3ACF926EE42F26CCEF)图标。

从DevEco Studio 6.0.2 Release版本开始，使用HarmonyOS Ask智能体分析代码文件。

以DevEco Studio 6.0.2 Release和DevEco Studio 6.0.1 Beta1版本为例说明，操作如下。

**DevEco Studio 6.0.2 Release版本**

1.  选择HarmonyOS Act智能体，在对话框中输入**@**符号选择**Files**，或点击**@****Add Context** > **Files**，选择需要分析的代码文件，或在对话框输入文件路径指定需要分析的代码文件，或分析当前代码文件。
2.  在对话框输入描述，点击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/6f/v3/6M_bYMCGRMy2Ou6civNiYg/zh-cn_image_0000002518721536.png?HW-CC-KV=V1&HW-CC-Date=20260313T040724Z&HW-CC-Expire=86400&HW-CC-Sign=FB5F9297FA0EA876FC10556166120864DE46B2C7CC01F635C57921A966E92EA0)发送后等待回复。

**DevEco Studio 6.0.1 Beta1版本**

分析代码文件：在对话框中输入**@**符号选择**Files**，或点击**@****Add Context** > **Files**，可对单个或多个代码文件进行分析。

分析代码片段：选中代码段，点击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/58/v3/-7lwc-TpRnKSZOiRCyHl0Q/zh-cn_image_0000002518561640.png?HW-CC-KV=V1&HW-CC-Date=20260313T040724Z&HW-CC-Expire=86400&HW-CC-Sign=8DC6F0FF670CDF62B7DCB408037280A9664988910C9DF0931DCE6CAAB56DE1D3)图标开启光标上下文功能，在对话框内描述，开始分析。
