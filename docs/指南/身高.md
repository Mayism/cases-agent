# 身高

此数据记录用户在某时刻的身高数据。

Harmony SDK类型常量：[samplePointHelper.height.DATA\_TYPE](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/health-api-samplepointhelper#section691019541405)

说明

Wearable设备暂不支持该数据类型。

## OAuth权限

联盟卡片申请的权限名称：健康数据 > 体脂数据

## 采样明细数据

### 明细字段说明
