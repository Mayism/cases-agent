# GetOriginShape

## 函数功能

## 函数原型

```cpp
Shape GetOriginShape() const;
```

## 参数说明

无

## 返回值

| 类型 | 描述 |
| --- | --- |
| [Shape](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cannkit-ge-shape) | TensorDesc描述的originShape。 |

## 异常处理

无
