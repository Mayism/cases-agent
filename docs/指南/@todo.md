# @todo

@todo 标签记录要完成的任务。在一个 ArkTSDoc 注释块中可以包含多个 @todo 标签。

## 语法

@todo text describing thing to do.

## 示例

使用 @todo 标签：

```javascript
/**
 * @todo Write the documentation.
 * @todo Implement this function.
 */
export function foo() {
  // write me
}
```
