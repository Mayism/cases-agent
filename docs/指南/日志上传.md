# 日志上传

说明

该功能仅支持中国境内（香港特别行政区、澳门特别行政区、中国台湾除外）。

若开发过程中遇到DevEco Studio卡顿、卡死或其他故障时，可点击IDE Error问题弹窗中Send Report，点击OK后向DevEco Studio回传日志信息。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/0/v3/sL_aYWCoS2Gm1mN1nJnE-Q/zh-cn_image_0000002500910700.png?HW-CC-KV=V1&HW-CC-Date=20260313T050511Z&HW-CC-Expire=86400&HW-CC-Sign=C8E5D9530C8D1B5179D542A0E7D361516FE5A7649890188F50FDF66AF6F7B6C4 "点击放大")

或通过菜单栏**Help > Collect Logs and Diagnostic Data**，选择并上传相关日志，帮助DevEco Studio提升稳定性体验。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/5f/v3/_VtxpxZhQaGNMAgrE3YvDQ/zh-cn_image_0000002532670611.png?HW-CC-KV=V1&HW-CC-Date=20260313T050511Z&HW-CC-Expire=86400&HW-CC-Sign=6029852AC68ABDC55D33436FEA322A1D7649C93B28C885EBBA9011C9BD3054CA "点击放大")

若开发者后续需要关闭数据采集功能，请在**File > Settings**（macOS为**DevEco Studio > Preferences/Settings**）**\> Appearance & Behavior > System Settings > Data Sharing**设置界面，取消勾选**Send usage statistics**关闭数据采集开关。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/93/v3/fAPfoCVOQ1G9kce3F1Zfqw/zh-cn_image_0000002532750565.png?HW-CC-KV=V1&HW-CC-Date=20260313T050511Z&HW-CC-Expire=86400&HW-CC-Sign=1416C46006D72632B38999C0239FB8E0BAEB2FF84627F6B1DD91AF6FB1573E9B)
