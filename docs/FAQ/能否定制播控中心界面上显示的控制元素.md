# 能否定制播控中心界面上显示的控制元素

播控中心的控制元素根据会话类型固定。

-   audio类型的控制元素：收藏，上一首，播放/暂停，下一首，循环模式。
-   video类型的控制元素：快退，上一首，播放/暂停，下一首，快进。
