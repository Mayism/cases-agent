# 如何申请多个长时任务

**问题现象**

应用在后台运行多个任务，需申请多个长时任务。

**解决措施**

同一时刻，一个UIAbility只能申请一个长时任务。需要创建多个UIAbility来申请不同种类的长时任务。不同时刻可以申请不同种类的长时任务。
