# HarmonyOS Webview如何实现透明效果

在HarmonyOS的Web组件中，可通过设置[opacity](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-opacity#opacity18)属性实现页面元素的透明效果。
