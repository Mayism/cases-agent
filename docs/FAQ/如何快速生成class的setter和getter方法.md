# 如何快速生成class的setter和getter方法

1.  在类内部，右键点击，选择“生成”。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/ba/v3/-lZPEvtySc6CDyCtWUynhA/zh-cn_image_0000002229758765.png?HW-CC-KV=V1&HW-CC-Date=20260313T030408Z&HW-CC-Expire=86400&HW-CC-Sign=7BA0D1740A759726A7CACF2D001B4FCED4168109E83CDC0702579081855456A0 "点击放大")
    
2.  选择Getter and Setter。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/0b/v3/5oTid29oQmiIp-IdGTTMyw/zh-cn_image_0000002229604269.png?HW-CC-KV=V1&HW-CC-Date=20260313T030408Z&HW-CC-Expire=86400&HW-CC-Sign=4BB124C1D4140250EEFF32A33E95A5E4A4CE97D668ADAAAB88470BE78B01E610 "点击放大")
    
3.  选择要生成Getters和Setters的字段，点击OK，快速生成。
