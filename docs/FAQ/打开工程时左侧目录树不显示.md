# 打开工程时左侧目录树不显示

**问题现象**

当删除工程目录下的.idea/modules文件夹或该文件夹不存在时，如下图所示。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/1/v3/FaRCbHQdRtOJm7hdSOqbKw/zh-cn_image_0000002229758589.png?HW-CC-KV=V1&HW-CC-Date=20260313T052646Z&HW-CC-Expire=86400&HW-CC-Sign=FBC2D39A02B9CD0DA9D252BEBB9D8BD5E8C4F489112CF4A34BECAEF2708B5CD0)

重新打开工程，左侧目录树未显示，如下图所示。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/44/v3/LxT8kEoESP6WBFJMpmKA5A/zh-cn_image_0000002194318328.png?HW-CC-KV=V1&HW-CC-Date=20260313T052646Z&HW-CC-Expire=86400&HW-CC-Sign=5364B30633BCE26F192231A2298582D61BF1958230C33A2996AC16AB05F9269D)

**问题原因**

modules文件夹下的iml文件定义工程模块结构信息，modules.xml定义模块结构文件的位置。删除modules文件夹后，无法找到对应的iml文件。

**解决措施**

关闭工程，删除文件管理器中的modules.xml文件，然后通过DevEco Studio重新打开工程，工程目录树即可恢复。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/57/v3/r3uiLGOdRoehSeWGlyZnRA/zh-cn_image_0000002194158720.png?HW-CC-KV=V1&HW-CC-Date=20260313T052646Z&HW-CC-Expire=86400&HW-CC-Sign=A0C7748EC0FAA4C6B3CB650283FF971060786B3AAECA6731CC41230B78245C17)
