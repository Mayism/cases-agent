# 如何解决编译报错“Error: the name 'XXX' can only contain [a-zA-Z0-9_]”的问题

**问题现象**

编译报错：文件名只能包含大小写字母、数字、下划线。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/53/v3/OTYtf4toSUKt69mB182ONQ/zh-cn_image_0000002194158636.png?HW-CC-KV=V1&HW-CC-Date=20260313T054144Z&HW-CC-Expire=86400&HW-CC-Sign=6DECB3BB2B9728D59CA0CDA318F73A57D28B31622245B1507C30F3BFA21E7181)

**解决方案**

检查文件名是否包含除了大小写字母、数字和下划线之外的其他字符。
