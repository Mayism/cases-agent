# 如何通过路由的方式打开半屏

统一使用导航组件（Navigation）进行操作，组件导航中参考页面显示类型的弹窗类型为 NavDestinationMode.DIALOG，DIALOG默认透明，进出路由栈不影响下层NavDestination的生命周期。
