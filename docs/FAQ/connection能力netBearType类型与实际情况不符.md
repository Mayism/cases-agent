# connection能力netBearType类型与实际情况不符

在双开场景下，即Wi-Fi和蜂窝网络同时连接时，netBearType参数仅返回Wi-Fi的状态信息。

参考文档：[NetBearType](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-net-connection#netbeartype)
