# 如何调用系统拍照并获取图片

可使用[Camera Picker](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-camerapicker)拉起系统相机进行拍摄，示例代码可参考[基于系统相机实现拍照功能](https://gitcode.com/HarmonyOS_Samples/camera-picker)。
