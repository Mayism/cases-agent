# 模拟器使用物理键盘无法输入中文

**问题现象**

点击输入框后，使用物理键盘无法输入中文。

**解决措施**

模拟器不支持物理键盘输入中文。如需输入中文，点击输入框时会弹出软键盘，使用软键盘进行输入。
