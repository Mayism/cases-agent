# 查看应用探索测试报告时，报告加载卡顿的原因是什么

更新时间: 2026-03-10 14:16

探索测试执行时间较长，会产生大量报告数据。当报告节点过多时，可能会出现报告资源加载卡顿。此时，可以关闭客户端并重启以尝试恢复。
