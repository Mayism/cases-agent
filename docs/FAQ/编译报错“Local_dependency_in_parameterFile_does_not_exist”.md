# 编译报错“Local dependency in parameterFile does not exist”

**错误描述**

parameterFile 中的本地依赖项不存在。

**可能原因**

parameterFile中定义的值不存在。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/ca/v3/XEuXqHwoSxWUMYmysJW0gg/zh-cn_image_0000002194318024.png?HW-CC-KV=V1&HW-CC-Date=20260313T054414Z&HW-CC-Expire=86400&HW-CC-Sign=77865287B107D11277D86A2272657DA39BF9EC23B181CBEBAB977DF8623A547E)

**解决措施**

检查本地依赖路径是否存在，以及是否区分大小写。
