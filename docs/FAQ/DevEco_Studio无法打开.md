# DevEco Studio无法打开

**问题现象**

在Windows 10和Windows 11中，修改字符编码后，安装在中文目录下的DevEco Studio无法打开，报错“Error launching...”。

**解决措施**

请在英文目录下重新安装DevEco Studio。
