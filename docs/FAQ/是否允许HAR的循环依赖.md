# 是否允许HAR的循环依赖

不允许循环依赖。可以把公共部分放到一个共享包中，或者使用[HAR模块间动态import依赖解耦](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/arkts-dynamic-import#har模块间动态import依赖解耦)。
