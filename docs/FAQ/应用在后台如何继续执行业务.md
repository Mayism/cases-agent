# 应用在后台如何继续执行业务

应用切换至后台时，可申请短时任务延迟挂起或长时任务避免挂起。

[短时任务](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/transient-task)，[长时任务](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/continuous-task)
