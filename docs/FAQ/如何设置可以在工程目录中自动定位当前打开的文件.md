# 如何设置可以在工程目录中自动定位当前打开的文件

**问题现象**

打开文件时，左侧工程目录自动定位到当前文件位置。

**解决措施**

在工程目录右上角点击Select Opened File图标，定位到当前打开的文件在工程目录中的位置。
