# 在投播过程中，出现设备连接失败、启动缓慢以及回调进度卡顿等问题，可能原因是什么

更新时间: 2026-03-10 14:16

造成此问题的原因可能是网络质量较差。请检查两端设备的网络连接情况，并尝试切换到网络条件更好的环境进行投播。
