# 如何使用DevEco Studio中的ArkTS代码模板

使用代码模板可以提高开发效率。在DevEco Studio中，可以通过“File \\> Settings \\> Editor \\> Live Templates”查看现有模板并新增自定义模板。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/4b/v3/KyI484WeTZehBlW1AP42Iw/zh-cn_image_0000002194318448.png?HW-CC-KV=V1&HW-CC-Date=20260313T052508Z&HW-CC-Expire=86400&HW-CC-Sign=76A654B90A71A1B3A38B4D10CBE306E82C5DB8781928D71098E1D3B5BB6B6437 "点击放大")

点击展开按钮查看现有模板详情，点击右上角加号新增自定义模板。

    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/69/v3/LSVYzhDWSCicqcWWc9UGwQ/zh-cn_image_0000002229604229.png?HW-CC-KV=V1&HW-CC-Date=20260313T052508Z&HW-CC-Expire=86400&HW-CC-Sign=A515FF18514AF55CDA967B0588B7D4E07E59545E780B19503CC8D68B95CC1AE4 "点击放大")
    
    选中自定义群组，点击+，新增模板。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/19/v3/CLNa6vdMRc2fLo1L2HngEg/zh-cn_image_0000002194318456.png?HW-CC-KV=V1&HW-CC-Date=20260313T052508Z&HW-CC-Expire=86400&HW-CC-Sign=8E993330CCE63A720E67146259FBD463B1CDC68753DD399961777D2F10E43076 "点击放大")

    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/ce/v3/87mUOhzeT9CcjOGb2nFyxg/zh-cn_image_0000002229604225.png?HW-CC-KV=V1&HW-CC-Date=20260313T052508Z&HW-CC-Expire=86400&HW-CC-Sign=818187EEA3A2763F4333B1D38E5B13EF8672353C61B5E3BD219EFD2633678F79 "点击放大")
    
    点击上图的“Change”处，选择代码模板文件。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/0b/v3/Ns4J7KGYTjqkoovD1Y-rHg/zh-cn_image_0000002194158856.png?HW-CC-KV=V1&HW-CC-Date=20260313T052508Z&HW-CC-Expire=86400&HW-CC-Sign=03253E0FA11CBA24CCE77A36BF5B569C5C95552A913E88E7D8BA181640B99A80 "点击放大")

    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/e8/v3/Y-ffkQNOTwOGgnYPCp3ovA/zh-cn_image_0000002229604237.png?HW-CC-KV=V1&HW-CC-Date=20260313T052508Z&HW-CC-Expire=86400&HW-CC-Sign=43523CBD4C421735F6D47B02646F126BE83C2421F9A3CB710011A7F0B749CFE8)
