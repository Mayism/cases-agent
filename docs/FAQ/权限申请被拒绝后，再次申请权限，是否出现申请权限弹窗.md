# 权限申请被拒绝后，再次申请权限，是否出现申请权限弹窗

通过设置页关闭权限后，再次申请该权限不会再次弹窗，返回值为-1。对于申请授权的返回值请参见：[PermissionRequestResult](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-permissionrequestresult)
