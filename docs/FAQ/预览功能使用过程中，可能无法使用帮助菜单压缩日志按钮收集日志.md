# 预览功能使用过程中，可能无法使用帮助菜单压缩日志按钮收集日志

**问题现象**

当同时预览多个工程时，点击帮助菜单中的“压缩日志”按钮，可能会因日志文件被占用而无法压缩。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/e7/v3/HF69fPLMQcu2-pzPY8GioA/zh-cn_image_0000002229603945.png?HW-CC-KV=V1&HW-CC-Date=20260313T052907Z&HW-CC-Expire=86400&HW-CC-Sign=081979A80FC52323262B163B76495C53D4FFE989D206901047A11A516EDA573A)

此时右下角会出现压缩失败的提示：

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/8d/v3/OOd2CZuqQomPVdULT4ovbA/zh-cn_image_0000002229758417.png?HW-CC-KV=V1&HW-CC-Date=20260313T052907Z&HW-CC-Expire=86400&HW-CC-Sign=0C9B0A5D3305CC56379D849AE7AC999C84EA3A7B7D83CDDF11FA5630427267EE)

**解决措施**

请关闭预览过的工程，或者重启DevEco Studio后不要打开预览器，即可再次压缩收集日志。
