# 运行UITest用例，运行超时屏幕黑屏后报错

**问题现象**

运行UITest用例时，如果屏幕黑屏并超时，将报错提示“Tests failed”。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/b7/v3/QwhBwwlyTGiV6StIqy7VpA/zh-cn_image_0000002229758661.png?HW-CC-KV=V1&HW-CC-Date=20260313T060236Z&HW-CC-Expire=86400&HW-CC-Sign=B7FB7BF4969BB3B740FF4C92381738FAB1D28BF8CADE8482896C63A3B2B509D1)

**解决措施**

运行UITest用例时，需确保设备屏幕常亮。屏幕常亮时，用例能够成功运行。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/70/v3/QBIiBsFsS8KbR1V_ca2X8Q/zh-cn_image_0000002194158788.png?HW-CC-KV=V1&HW-CC-Date=20260313T060236Z&HW-CC-Expire=86400&HW-CC-Sign=826AD16CDE9CBEE4FC11425FE92D55A1DA9D4F465DF862F126DBADC397996EA2)
