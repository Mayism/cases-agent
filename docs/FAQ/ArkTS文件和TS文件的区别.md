# ArkTS文件和TS文件的区别

ArkTS基于TS语法，兼容TS的大部分特性。ArkTS扩展了声明式UI能力，使开发者能够以更简洁、自然的方式开发高性能应用。推荐用ArkTS开发UI，用TS开发业务逻辑。
