# 模拟器启动后，设备无法识别

**问题现象**

场景一：调试运行时，如果安装HAP失败，提示“Device not found or connected”，请检查设备是否已正确连接。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/b2/v3/J6x--BovQVu0dfRBl1F5Xg/zh-cn_image_0000002229603797.png?HW-CC-KV=V1&HW-CC-Date=20260313T055049Z&HW-CC-Expire=86400&HW-CC-Sign=2BC5D746E54E94C036E9C294A4404E49C517F6790AEEDC76F7A9C4E6E503FD51)

场景二：DevEco Studio无法识别已连接的设备，显示“No device”。

**原因**

hdc工具的进程或模拟器存在问题。

**解决措施**

1.  执行以下命令，终止hdc进程，然后重新连接。
    
    ```bash
    hdc kill
    ```
    
2.  若按照步骤1操作后仍无法连接，请重启模拟器，然后重新尝试连接。
