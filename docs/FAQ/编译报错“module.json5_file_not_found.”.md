# 编译报错“module.json5 file not found.”

**错误描述**

未找到module.json5文件。

**可能原因**

模块下的module.json5文件缺失。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/95/v3/UdufUoilRRCwHVVCStVpSA/zh-cn_image_0000002229604093.png?HW-CC-KV=V1&HW-CC-Date=20260313T054448Z&HW-CC-Expire=86400&HW-CC-Sign=B120876CD7BE7B14B2FD9D8FC915E5A10A96670AC195C17C6D6680A664F3AD9D)

**解决措施**

在Stage模型项目中，确保每个模块包含module.json5文件。
