# 使用方舟异常信息增强检测

## 概述

在进行ArkTS项目开发中可能存在需要加载native模块的场景，开启方舟native模块加载异常信息增强功能后，可以丰富ArkTS项目中因加载native模块导致的报错信息，以便更准确地进行native问题定位。

## 启用方舟native模块加载异常信息增强

可以通过以下两种方式启用方舟native模块加载异常信息增强

-   方式一
    
    点击**Run > Edit Configurations >** **Diagnostics**，勾选**Enhanced Error Info**。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/df/v3/B3QEfuAMSeKv5oaabZnvow/zh-cn_image_0000002404125161.png?HW-CC-KV=V1&HW-CC-Date=20260313T023928Z&HW-CC-Expire=86400&HW-CC-Sign=C748B1F1C5C858C83502448915F37F38BC92E21FEEFA99B7DCBD19AA162999C6)
    

-   方式二
    
    通过命令行开启。
    
    ```wasm
    aa start {abilityName} {bundleName} -E
    ```
    

## 启用方舟native模块加载异常信息增强

1.  运行或调试当前应用。
2.  当程序出现因native模块加载导致的报错信息时，会显示更详细准确的错误信息。
